# 嫂子站 · 代号 K

纯前端桌面式 ARG 网页原型。所有账号、密码、登录和文件内容均为游戏内虚构交互，不连接任何真实平台。

## 运行

直接打开 `index.html`，或在目录中启动任意静态文件服务器。

## 核心交互

- 双击桌面图标打开应用。
- 窗口支持拖动、缩放、最小化、关闭与层级切换。
- INS、KKTalk、微博、浏览器和文件夹共同推进调查状态。
- 调查进度保存在 `localStorage`。
- 六项关键证据完成后，`K_BACKUP` 自动解锁。
- 最终文件提供五种结局。
