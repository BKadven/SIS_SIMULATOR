# 嫂子站 · 代号 K

纯前端桌面式 ARG 网页原型。所有账号、密码、登录和文件内容均为游戏内虚构交互，不连接任何真实平台。

## 运行

直接打开 `index.html`，或在目录中启动任意静态文件服务器。

## 核心交互

- 每次打开网页先显示剧情序章，点击登录后进入 macOS 风格调查桌面。
- 双击桌面图标打开应用。
- 窗口支持拖动、缩放、最小化、关闭与层级切换。
- INS、KKTalk、微博、浏览器和文件夹共同推进调查状态。
- 调查进度保存在 `localStorage`。
- 收集 A / B / C 三条关键线索后，可使用三线密码解锁 `K_BACKUP`。
- 最终文件先验证老鬼身份；正确验证并联系安抚后，才有足够时间进入隐藏结局 5。
- 未发现或未安抚老鬼时，结局 1–4 正常；尝试结局 5 会强制转入结局 4。
